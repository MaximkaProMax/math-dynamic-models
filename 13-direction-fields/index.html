Теория

Ключевые идеи темы

Дифференциальное уравнение первого порядка задает не саму функцию, а наклон ее графика в каждой точке плоскости.

Поле направлений

В каждой точке рисуется короткий отрезок с наклоном y' = f(x, y).

Метод Эйлера

Решение строится последовательными шагами по текущему наклону.

Изоклины

Линии f(x, y)=k объединяют точки с одинаковым наклоном.

Интерактивная модель

Поле направлений, решение и изоклины

Выберите уравнение, диапазоны и начальную точку. Можно включить изоклины для нескольких значений наклона.

Рабочая идея

Поле помогает качественно понять поведение решений еще до точного интегрирования.

x_{n+1}=x_n+h, y_{n+1}=y_n+h f(x_n,y_n)
